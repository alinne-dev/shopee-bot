# Bot de Curadoria Inteligente

Sistema de curadoria automatizada para nicho de beleza e afiliados. O bot gerencia uma base de dados de produtos e realiza postagens agendadas de forma orgânica no Telegram.

## 🚀 O que este projeto faz

* **Gestão de Curadoria:** Processa uma lista de produtos estruturada em `produtos.json`, garantindo que o feed esteja sempre atualizado.
* **Automação Estratégica:** Realiza postagens agendadas em horários de pico, maximizando o engajamento.
* **IA de Conteúdo:** Utiliza IA para gerar legendas coloquiais e naturais, focadas em "indicação de amiga" em vez de vendas agressivas.
* **Infraestrutura Resiliente:** Sistema executado em container com lógica de reconexão automática e tratamento robusto de erros de API.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python
* **API:** API do Bot do Telegram
* **Implantação/Infraestrutura:** Railway
* **Formato de dados:** JSON
